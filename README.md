This repository contains all files, source code, and documentation for the AI-powered intrusion detection system developed as part of the COMP-699B course.

## Key Features
- Hybrid Machine Learning (Random Forest + MLP)
- Explainable AI using SHAP
- Flask-based visualization dashboard
- Google Colab environment

## How to Run
- Open `notebooks/AI_IDS_Google_Colab.ipynb` in Google Colab.
- Run each cell sequentially.
- View SHAP explanations and prediction outputs.

## Results
Accuracy: **93.6%**  
Average Detection Time: **1.8 seconds**
